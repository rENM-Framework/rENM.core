# rENM.core 0.1.0

* Initial release.
* Added `rENM_project_dir()`, which resolves the rENM project root from an
  explicit argument, R option, or environment variable.
* Added `get_species_info()` to retrieve standardized metadata for a species
  by alpha code.
* Added `show_species()` to inspect the species metadata table for a project.
* Added `show_variables()` to inspect the environmental variables metadata
  table for a project.
